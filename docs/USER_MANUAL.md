# Manual de Usuario - CRM con Clientes y Notas

## ¿Qué es esta aplicación?

Esta aplicación es un sistema simple para gestionar clientes y sus notas. Te permite agregar información de clientes y escribir notas sobre cada uno de ellos.

## ¿Cómo acceder?

Ingresa a: https://app-qpa1m-6cdk6hvao-luisernestosalas-2775s-projects.vercel.app

## Funciones principales

### 1. Ver clientes
- Al abrir la aplicación verás una lista de todos tus clientes
- Cada cliente muestra su nombre y información básica

### 2. Agregar un cliente nuevo
1. Busca el botón "Agregar Cliente" o "+"
2. Completa los campos requeridos:
   - Nombre del cliente
   - Información de contacto (teléfono, email, etc.)
3. Haz clic en "Guardar"

### 3. Editar información de cliente
1. Haz clic sobre el cliente que quieres editar
2. Modifica la información necesaria
3. Guarda los cambios

### 4. Agregar notas a un cliente
1. Selecciona el cliente
2. Busca la sección de "Notas"
3. Escribe tu nota en el campo de texto
4. Guarda la nota
