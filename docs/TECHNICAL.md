# Documentación Técnica - CRM App

## Información General

**Nombre:** CRM con clientes y notas  
**Stack Tecnológico:** HTML + JavaScript  
**Repositorio:** https://github.com/luisernestosalas/app-qpa1m  
**URL de Producción:** https://app-qpa1m-6cdk6hvao-luisernestosalas-2775s-projects.vercel.app  

## Descripción

Aplicación web de gestión de relaciones con clientes (CRM) desarrollada con tecnologías frontend básicas. Permite administrar información de clientes y mantener notas asociadas a cada uno.

## Tecnologías Utilizadas

- **Frontend:** HTML5, JavaScript ES6+
- **Hosting:** Vercel
- **Control de versiones:** Git/GitHub

## Funcionalidades Principales

### Gestión de Clientes
- Registro de nuevos clientes
- Visualización de lista de clientes
- Edición de información de clientes
- Eliminación de registros de clientes

### Sistema de Notas
- Creación de notas asociadas a clientes
- Visualización de notas por cliente
- Edición de notas existentes
- Eliminación de notas
