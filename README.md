# CRM con Clientes y Notas

Una aplicación web simple para gestión de relaciones con clientes (CRM) que permite administrar información de clientes y sus notas asociadas.

## 🚀 Demo

**URL de la aplicación:** [https://app-qpa1m-6cdk6hvao-luisernestosalas-2775s-projects.vercel.app](https://app-qpa1m-6cdk6hvao-luisernestosalas-2775s-projects.vercel.app)

## 📋 Características

- ✅ Gestión completa de clientes (CRUD)
- ✅ Sistema de notas por cliente
- ✅ Interfaz responsive y moderna
- ✅ Almacenamiento local en el navegador
- ✅ Búsqueda y filtrado de clientes
- ✅ Diseño intuitivo y fácil de usar

## 🛠️ Tecnologías

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Almacenamiento:** LocalStorage del navegador
- **Despliegue:** Vercel
